# Features
這是一個使用 Express 製作的簡易網頁應用，內容包含4個頁面：

- 首頁
- About
- Portfolio
- Contact

# Installation
### 安裝npm套件以及nodemon套件
```
1. npm install // 安裝npm套件
```

```
2. npm install -g nodemon // 安裝nodemon套件
```
### 執行app.js啟動伺服器

```
3. nodemon app.js
```

### 開啟瀏覽器輸入 http://localhost:3000 即可進入express_basic_WebServer
